Get OS X Mail to sort Exchange Folders Alphabetically

Posted on Mon 14 November 2016 in How To

Original source here

Newly created folders are not sorted alphabetically to sort this out:

  1. Quit Mail app
  2. Run the following command in a terminal cp /Users/<username>/Library/Mail/V3/<mailbox>/.mboxCache.plist{,.bck}
  3. Restart Mail check if folder are displaying correctly and all working.
  4. Remove .bck file
